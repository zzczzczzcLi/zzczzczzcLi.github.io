Linux常用命令

2023/5/6 教程 Linux

主机上复制的内容可以出现在虚拟机上，前提是虚拟机内没有使用过复制。（Ctrl+C，Ctrl+V）
主机文件直接复制进虚拟机会不完整，要用ftp。

ls（list）：显示当前文件夹所有内容，文件夹名字为蓝色
tab：如果你当前打的命令是唯一的，那么就会给你补全，例如输入fir+tab会补全成firefox

open：打开某个文件，也可以是文件夹。（可以使用[open Li*]来打开所有前缀都是Li的文件）

1. 这里需要注意的是，查询固定后缀时要带上文件格式，例如open *Li.md，如果不明确文件格式，就写成open *Li\*
2. 同时open语句也可以用来打开URL，例如open http://localhost

firefox：安装了火狐浏览器的话就可以使用这个命令，firefox baidu.com或者firefox localhost
rm（remove）：删除某个内容，文件要带上后缀
mv（move）：移动，文件->目的地，都是路径

cd（change directory）：变动文件目录

1. cd <文件夹>，进入当前文件夹的下级文件夹
2. cd -，返回上一步操作（只记忆一步）
3. cd，返回桌面根目录
4. cd ..，返回上级目录（..代表上级，.代表当前目录，例如是打开当前路径）

clear：清除终端命令记录
date：显示当前时间
ctrl+c：终止并退出程序（没有程序就不会退出）
ctrl+d：退出终端
ctrl+alt+t：打开一个终端（终端的左上角的‘+’可以直接新建终端，并且是放在一起的）

find：寻找

find：会显示当前文件夹下的所有内容，和ls的区别是，ls会标记文件夹为蓝色。
find [file name]：显示当前文件夹内所有符合要求的文档（通配符在Linux中均可用），例如find *001\*，只要文件名包含001都会被显示出来。（注意带后缀）

mkdir（make directoy）：新建文件夹

vim（visual editor improved）：新建文件，例如vim hello.md。

1. 刚进入是命令行模式，:q是退出，:w是写入（注意冒号要打出来）
2. 按i进入插入模式，正常写入
3. 按esc退出到命令行模式

poweroff：关机
reboot：重启
