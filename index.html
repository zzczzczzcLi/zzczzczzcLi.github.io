机场推荐（置顶）
工具 2024/11/18
前言

一想到好兄弟们在用盗版翻墙软件我就难过，比如下面这个。

这东西的存在就和steam对应steam管家一样。
他们可能是从广大机场那买了订阅过来，然后封包成APP美名其曰是自己的机场。
很有可能是人家低买高卖，骗骗首次翻墙的人。

机场速度通常情况下：直连<中转（BGP）<专线（IPLC）
并且似乎现在国内防火墙的力度需要每天更新一次订阅。
一般情况下直连 5r能买到1T不限时流量是正常的
中转大约在10r/200g左右，若有浮动 一般都是机场质量偏差，或者有概率跑路

那些APP在做什么：（选择这一步的通常会亏钱，但是能用）
1.找了一个Clash,Clash Meta,Sing-box,Nekobox类似的软件
2.加入订阅
3.修改一下前端，变成某某加速器某某vpn

正常应该做什么：
1.找了一个Clash,Clash Meta,Sing-box,Nekobox类似的软件
2.加入订阅

现在不推荐使用Clash系列的，Clash开发人都跑路了，软件没人维护，但是网络协议日新月异。
比如Hysteria协议就不支持。

电脑端：推荐使用Nekoray
手机端：推荐使用Nekobox
苹果：各大机场首页都有教程，有些会有共享ID，可以用来下载ios专用加速器，比如Shadowrocket

如果网络特别差，订阅更新不了，可以加入Hysteria2核心（此处自行研究）
手机上是Hysteira2插件（全平台通用的）
订阅可能需要先挂梯子才能更新（如果没有梯子，请使用奶昔订阅转换）
转换的作用就是帮你拿订阅，顺带提升一下安全等级，然后换一个格式给你可以不用挂梯子就能用的订阅。

直连机场推荐

哈基米
量非常大，6r/1T不限时，刷图片可以，刷视频跳着加载大约等7-8秒
如果要用来刷视频，需要提前缓存好

三三云
和上家差不多，但是跳着刷视频只需要等3-5秒，稍微好一点点

性价比机场
很老牌的机场了，有1r/50g包月套餐，跳着刷基本不用等，最多1-2秒

using namespace std;

map<double,string>f;

    double chuxu,yuni,fuji;

    chuxu=80;

    for(auto i:f){
        cout<<setprecision(3)<<i.first<<" "<<i.second<<"\n";
    }
    return 0;
}

面包板使用前注意
教程 2025/1/25

要先用镊子顶开铁夹子，要接哪些口都要用镊子先捅进去（诗人我吃）

一些单词：
IPU，InputPullUp，输入模式且默认为高电位（上拉电阻），若按钮按下是导通，则接低电位，会导致引脚为低电位，以及IPD类似作用。
OD，OpenDrain，开漏
PP，PushPull，推挽，Pull-Up，Pull-Down，拉高（电平），拉低（电平），可以选择低电平或者高电平。
和上面开漏不同的是，开漏只能漏东西（即低电平），没有高电平，高电平则为高阻抗（断路）

tip：
写代码的时候，如果用了RCC_开头的函数，那么后面的参数也是RCC开头的
同理GPIO也是

接上一次任务，解析会了之后，本次较为简单：
能够PC13点灯之后，使用面包板接线，实现点亮发光二极管，流水灯（循环一个一个点亮），蜂鸣器的使用
此处循环的时候 1<<i，就是具体pin脚，比如GPIO_Pin_12，就是1<<i，赋值给GPIO结构体的成员PIN就可以了
具体看define
此处修改引脚的speed，并不会导致蜂鸣器声音变化，引脚speed只决定引脚的驱动能力上限和信号传递速度
并不直接影响电流大小。

元素之诗保存配置与快速进入服务器
游戏 2025/1/24

众所周知MC启动游戏是直接启动java，但是网易mc会先更新客户端
实际上是把某些文件夹全部删除了，然后再重新下载一遍，元素之诗材质大概有600MB，网速差的人，启动就会很慢（有位QQ好友就是）
并且，他删除的配置，比如材质包，按键设置，全给你删了，启动每次都要改。

所以如果只玩一个服务器的情况下，可以按下面步骤来进行设置。

  1. 进入游戏（注意下面的操作都不要退出游戏）
  2. 进行你所有的配置：ESC里面的全部设置（游戏内的是否显示伤害数字，以及数字按键1队友发光，F12时装是否显示，这个无法保存）
  3. 返回至大区传送大厅
  4. 添加自定义规则，禁止修改和删除，具体文件在下面（用你的杀毒软件，或者火绒）


    当然版本更新的时候，材质增加了出现问题，那就需要让他更新，把规则先关闭，启动游戏，再按照上面步骤来一遍。
