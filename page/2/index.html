机场推荐
工具 2024/11/18
前言

一想到好兄弟们在用盗版翻墙软件我就难过，比如下面这个。

这东西的存在就和steam对应steam管家一样。
他们可能是从广大机场那买了订阅过来，然后封包成APP美名其曰是自己的机场。
很有可能是人家低买高卖，骗骗首次翻墙的人。

机场速度通常情况下：直连<中转（BGP）<专线（IPLC）
并且似乎现在国内防火墙的力度需要每天更新一次订阅。
一般情况下直连 5r能买到1T不限时流量是正常的
中转大约在10r/200g左右，若有浮动 一般都是机场质量偏差，或者有概率跑路

那些APP在做什么：（选择这一步的通常会亏钱，但是能用）
1.找了一个Clash,Clash Meta,Sing-box,Nekobox类似的软件
2.加入订阅
3.修改一下前端，变成某某加速器某某vpn

正常应该做什么：
1.找了一个Clash,Clash Meta,Sing-box,Nekobox类似的软件
2.加入订阅

现在不推荐使用Clash系列的，Clash开发人都跑路了，软件没人维护，但是网络协议日新月异。
比如Hysteria协议就不支持。

电脑端：推荐使用Nekoray
手机端：推荐使用Nekobox
苹果：各大机场首页都有教程，有些会有共享ID，可以用来下载ios专用加速器，比如Shadowrocket

如果网络特别差，订阅更新不了，可以加入Hysteria2核心（此处自行研究）
手机上是Hysteira2插件（全平台通用的）
订阅可能需要先挂梯子才能更新（如果没有梯子，请使用奶昔订阅转换）
转换的作用就是帮你拿订阅，顺带提升一下安全等级，然后换一个格式给你可以不用挂梯子就能用的订阅。

直连机场推荐

哈基米
量非常大，6r/1T不限时，刷图片可以，刷视频跳着加载大约等7-8秒
如果要用来刷视频，需要提前缓存好

三三云
和上家差不多，但是跳着刷视频只需要等3-5秒，稍微好一点点

性价比机场
很老牌的机场了，有1r/50g包月套餐，跳着刷基本不用等，最多1-2秒

破解机房的电脑控制
教程 2024/3/29

tip:如果已经被控制了，先物理拔掉网线，再执行下面操作，然后把网线插回去
直接用taskkill /f /im 强制杀掉进程，一般来说都是studentmain.exe，显示上方没有举手的那个框框
就行了，有些控制比较极端，教学楼的是student.exe，而且杀了还会自动打开。
那么，循环杀，不停的杀就行了，bat代码放在下面了。

@echo off
set var=0
rem ************循环开始了
:continue
set /a var+=1
echo 第%var%次循环
taskkill /f /im student.exe
if %var% lss 1000000000000000000000000000000000000000000000 goto continue
rem ************循环结束了
echo 循环执行完毕
pause
